# 🎯 One Target - فروشگاه تخصصی قطعات کامپیوتر

یک فروشگاه آنلاین کامل برای قطعات کامپیوتر با قابلیت‌های پیشرفته از جمله پنل مدیریت، سیستم احراز هویت، سبد خرید، علاقه‌مندی‌ها، سفارشات و دستیار هوش مصنوعی محلی.

---

## ✨ قابلیت‌ها

- 🔐 **سیستم احراز هویت** (ورود، ثبت‌نام، خروج)
- 🛒 **سبد خرید** کامل با مدیریت تعداد و حذف
- ❤️ **علاقه‌مندی‌ها** (Wishlist)
- 📦 **مدیریت سفارشات** با جزئیات کامل
- 👑 **پنل مدیریت ادمین** (مدیریت محصولات، سفارشات، کاربران)
- 🤖 **دستیار هوش مصنوعی محلی** با LM Studio
- ⚡ **شگفت‌انگیزه هوش مصنوعی** با تایمر تخفیف
- 📱 **طراحی ریسپانسیو** (مناسب برای موبایل و دسکتاپ)
- 🔍 **جستجو و فیلتر پیشرفته** محصولات
- 💳 **ثبت سفارش** با اطلاعات گیرنده

---

## 🛠️ تکنولوژی‌های استفاده شده

| بخش | تکنولوژی |
|-----|----------|
| **فرانت‌اند** | HTML5, CSS3, JavaScript (Vanilla) |
| **بک‌اند** | PHP 8.2+ (بدون فریمورک) |
| **دیتابیس** | MySQL 8.0+ |
| **وب‌سرور** | Apache (XAMPP / WAMP / Laragon) |
| **هوش مصنوعی** | LM Studio + Gemma 3n 6.9B |

---

## 📋 نیازمندی‌ها

برای اجرای این پروژه نیاز به موارد زیر دارید:

### 1️⃣ نرم‌افزارهای ضروری

| نرم‌افزار | نسخه | توضیح |
|-----------|------|--------|
| **XAMPP / WAMP / Laragon** | آخرین نسخه | برای اجرای Apache و MySQL |
| **PHP** | 8.2 یا بالاتر | زبان بک‌اند |
| **MySQL** | 8.0 یا بالاتر | دیتابیس |
| **مرورگر** | Chrome / Firefox / Edge | برای باز کردن سایت |
| **LM Studio** | آخرین نسخه | برای هوش مصنوعی (اختیاری) |

### 2️⃣ اکستنشن‌های PHP (فعال بودن در php.ini)
