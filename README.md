یک فروشگاه آنلاین کامل برای قطعات کامپیوتر با قابلیت‌های پیشرفته از جمله پنل مدیریت، سیستم احراز هویت، سبد خرید، علاقه‌مندی‌ها، سفارشات و دستیار هوش مصنوعی محلی.

---

## ✨ قابلیت‌ها

- 🔐 سیستم احراز هویت (ورود، ثبت‌نام، خروج)
- 🛒 سبد خرید کامل با مدیریت تعداد و حذف
- ❤️ علاقه‌مندی‌ها (Wishlist)
- 📦 مدیریت سفارشات با جزئیات کامل
- 👑 پنل مدیریت ادمین (مدیریت محصولات، سفارشات، کاربران)
- 🤖 دستیار هوش مصنوعی محلی با LM Studio
- ⚡️ شگفت‌انگیزه هوش مصنوعی با تایمر تخفیف
- 📱 طراحی ریسپانسیو (مناسب برای موبایل و دسکتاپ)
- 🔍 جستجو و فیلتر پیشرفته محصولات
- 💳 ثبت سفارش با اطلاعات گیرنده

## 🚀 راه‌اندازی پروژه (گام‌به‌گام)

### مرحله ۱: دانلود و نصب XAMPP

از سایت Apache Friends XAMPP را دانلود و نصب کنید.

بعد از نصب، Apache و MySQL را روشن کنید.

### مرحله ۲: قرار دادن پروژه در مسیر درست

پروژه را در مسیر زیر قرار دهید:

C:\xampp\htdocs\one-target\     (ویندوز)
/opt/lampp/htdocs/one-target/   (لینوکس)
/Applications/XAMPP/htdocs/one-target/ (مک)

### مرحله ۳: ایجاد دیتابیس

1. مرورگر را باز کنید و به آدرس زیر بروید:
   http://localhost/phpmyadmin

2. روی New کلیک کنید و یک دیتابیس جدید به نام one_target_db بسازید
   Collation: utf8mb4_persian_ci

3. روی تب Import کلیک کنید و فایل database.sql را آپلود کنید

4. روی Go کلیک کنید

📌 حساب ادمین پیش‌فرض:
ایمیل: admin@onetarget.com
رمز عبور: password

### مرحله ۴: پیکربندی فایل‌ها

مسیر فایل‌ها را در صورت نیاز تغییر دهید:

در js/main.js (خط ۱):
const API_URL = '/one-target/backend/index.php';

در backend/index.php (خط ۱۶-۱۹):
$host = 'localhost';
$dbname = 'one_target_db';
$username = 'root';
$password = '';  // اگر رمز داری، اینجا وارد کن

### مرحله ۵: اجرای سایت

مرورگر را باز کنید و به آدرس زیر بروید:

http://localhost/one-target/index.html

## 🤖 راه‌اندازی هوش مصنوعی (اختیاری)

### ۱. دانلود و نصب LM Studio

از سایت LM Studio آخرین نسخه را دانلود و نصب کنید.

### ۲. دانلود مدل

1. LM Studio را باز کنید
2. به برگه Search بروید
3. مدل gemma-3n-e4b-instruct-6.9b را جستجو و دانلود کنید
4. مدل را Load کنید

### ۳. روشن کردن سرور LM Studio

1. در LM Studio، به برگه Local Server بروید
2. روی Start Server کلیک کنید
3. آدرس سرور: http://127.0.0.1:1234
